关于MarkDown的使用指南

MarkDown标题显示

使用 = 和 - 标记

通过在文字下方添加“=”为文字添加一级标题 ，通过“-”号为文字提供二级标题
（无法在已经使用Html语句的情况下对文字进行标题添加）

使用 # 号标记

当添加标题区分过多时可以通过在文字前方添加#号为文字添加标题，通过控制添加"#"号数量来区分标题级别（最多支持到6级标题，6及以上将以文本新式原样输出）

MarkDown字体显示

粗体，斜体，粗斜体

在文本的开头于结尾添加"_"和"*"来设置文本格式，通过控制"_"和"*"的数量来选择对应文本格式的添加类型

删除线

在文本的开头与结尾添加"~~"来为文本添加删除线

下划线

MarkDown未定义下划线相关语法，如果要使用的话需要使用HTML语法和CSS语法为文字添加下滑线

分割线

在MarkDown语法里通过大于3个以上重复的"*","-","_"来构建分割线。在使用"-"分割线时，前面如果有文字的话，间隔一行再去处理，否则编译器会将他识别为标题。
也可以同过HTML语句中的<hr>来添加分割线

MarkDown列表显示

无序列表

通过在文本前方添加"-"，"+","*"使得文本成为无序列表。列表嵌套时在次级列表开头添加四个空格即可

有序列表

通过在文本前方添加数字并以及"."使得文本成为有序列表。列表嵌套时在次级列表开头添加四个空格即可

MarkDown区块显示

在段落开头使用">"符号来标记区块，通过">"符号的数量来对区块进行嵌套。区块可以与列表组合使用

MarkDown代码显示

文本中添加

对段落中存在的文本通过"`"包住，即可使用。

代码区块

通过` ``` `包围代码块来显示，可以指定语言

MarkDown链接显示

通过[链接名称](链接地址)的形式构建链接，也可以通过"<>"包裹链接，当然直接放置也可以正常使用

MarkDown图片显示

通过![替代文字(图片无法加载时使用)](链接地址 "可选标题(加上等于创建Title)")的形式构建链接

MarkDown表格显示

通过使用`|`来分隔不同的单元格，使用`-`来分隔表头和其他行,在符号`-`前头添加`:`来实现左对齐，在符号`-`后头添加`:`来实现右对齐，在符号`-`前后添加`:`来实现居中。
